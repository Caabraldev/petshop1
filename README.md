# Petshop Mundo Animal 🐾
Projeto desenvolvido para a disciplina de **Fundamentos Web**.

## 📌 Descrição

Este projeto de um sistema web simples para um petshop, com o objetivo de apresentar informações sobre o estabelecimento, seus produtos e serviços.

## 🛠️ Tecnologias utilizadas

* HTML

## 📁 Estrutura do projeto

```
petshop-mundo-animal
│
├── pages
│   └── index.html
│
└── README.md
```

## 📄 Conteúdo do site

O sistema apresenta:

* Produtos para pets (rações e brinquedos)
* Informações do petshop
* Rodapé com dados de contato
* Navegação simples
